# x402 Facilitators Page

Facilitators registry page served at https://facilitators.x402.watch/

## Files

- `index.html` - Main page with facilitators list
- `favicon.ico` - KAMIYO favicon
- `og-image.png` - Open Graph image for social media

## Deployment

Served via Render from kamiyo website public directory. Updates are automatic on push to main branch.

## Metadata

Full Open Graph and Twitter Card metadata included for social media previews.
