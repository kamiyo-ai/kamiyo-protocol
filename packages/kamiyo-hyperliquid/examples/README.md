# Examples

Basic examples demonstrating how to use the Hyperliquid Security Monitor.

## Basic Monitoring

`basic_monitoring.py` - Simple example showing how to:
- Initialize a monitor
- Fetch security events
- Handle results

Run:
```bash
cd examples
python3 basic_monitoring.py
```

## Requirements

Make sure the parent directory is in your Python path, or run from the examples directory as shown above.

All dependencies from `requirements.txt` must be installed.
